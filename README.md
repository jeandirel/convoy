# convoy
Un convoyeur qu'on voit. Je m'appelle Minko Mambi Franck, fondateur de GastyConvoy. Basé à Maromme (76), j'ai créé cette micro-entreprise avec une conviction simple : confier sa voiture à quelqu'un, ça ne se prend pas à la légère.  Chaque convoyage que j'effectue, je le traite comme si la voiture était la mienne.
